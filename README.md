# 🎬 Movie Database App

A modern movie database web application built with React that allows users to browse popular movies, search films, and view detailed information using The Movie Database (TMDB) API.

---

## 🚀 Features

- 🔍 Search movies by title
- 🎞️ View popular & trending movies
- 📄 Movie detail page (overview, rating, release date, etc.)
- 🎬 Trailer integration (YouTube)
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly design

---

### 🚧 In Progress

- ❤️ Favorite system (requires authentication setup)
- 👤 User login / session management improvement

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- TMDB API
- Tailwind CSS (or your styling system)
- TypeScript (adjust if needed)
